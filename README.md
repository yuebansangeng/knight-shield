## About

* 快速调试，配置文件按需添加（单个js即可调试）
* 批量编译（libs）
* 在项目统一管理多组件（批量调试，批量发布）

## Getting Started

```sh
$ npm install @beisen/storybook-lib --save-dev
$ npx sbl storybook start
```
